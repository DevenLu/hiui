# hiui-v4

## 安装依赖

```sh
yarn
```
## 为每个包安装依赖

```sh
yarn bootstrap
```

## 预编译

```sh
yarn build
```

## 开始开发

```sh
yarn storybook
```

## 创建组件

> 注意小写中环线命名规范

```sh
yarn create:pkg ui `component-name`
```

## 创建通用工具函数

> 注意小写中环线命名规范

```sh
yarn create:pkg util `util-name`
```

## 创建自定义 hooks

> 注意小写中环线命名规范

```sh
yarn create:pkg hook `hook-name`
```
