# EmptyState 动效加载

用于请求和数据加载成功的中间状态，缓冲用户的等待焦虑

## 何时使用

页面加载时

页面卡片、浮层面板加载时

## Props

| 参数    | 说明                   | 类型                | 可选值                          | 默认值    |
| ------- | ---------------------- | ------------------- | ------------------------------- | --------- |
| size    | EmptyState 的尺寸         | string              | 'large' \| 'default' \| 'small' | 'default' |
| label   | 自定义加载中状态的文案   | React.ReactNode     | -                               | -         |
| active  | 是否显示 EmptyState       | boolean             | true \| false                   | true      |
| full    | 是否全屏               | boolean             | true \| false                   | false     |

## Methods

`EmptyState.open(target, { content, size, duration, key })`

| 参数     | 说明                                          | 类型                | 可选值                          | 默认值    |
| -------- | --------------------------------------------- | ------------------- | ------------------------------- | --------- |
| key      | 标识 `EmptyState` 的唯一 key，仅用于关闭         | string              | -                               | -         |
| target   | 需要 `EmptyState` 遮罩的元素, 当不传时为全屏遮罩 | HTMLElement         | -                               | -         |
| label   | 自定义加载中状态的文案                        | string \| ReactNode | -                               | -         |
| delay   | EmptyState 自动关闭的时间，单位为毫秒            | number              | -                               | -         |
| size     | EmptyState 的尺寸                                | string              | 'large' \| 'default' \| 'small' | 'default' |

`EmptyState.close(key)`

| 参数 | 说明                      | 类型   | 可选值 | 默认值 |
| ---- | ------------------------- | ------ | ------ | ------ |
| key  | 标识 `EmptyState` 的唯一 key | string | -      | -      |
