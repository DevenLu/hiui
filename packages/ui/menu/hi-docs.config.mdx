## Events

| 名称           | 说明                 | 类型                                                                 | 参数                                                    | 返回值 |
| -------------- | -------------------- | -------------------------------------------------------------------- | ------------------------------------------------------- | ------ |
| onClick        | 点击菜单选项时的回调 | (activeId: string \| number, prevActiveId: string \| number) => void | activeId: 激活的 ID <br/> prevActiveId: 上一次激活的 ID | -      |
| onClickSubMenu | 点击父菜单项时的回调 | (subMenuIndexs) => void                                              | subMenuIndexs: 当前点击菜单的索引数组                   | -      |
| onCollapse     | 点击收缩开关时的回调 | (collapsed: boolean) => void                                         | collapsed: 打开状态                                     | -      |

## Type

### DataItem

| 参数     | 说明                                             | 类型                | 可选值        | 默认值 |
| -------- | ------------------------------------------------ | ------------------- | ------------- | ------ |
| content  | 菜单项标题                                       | string \| ReactNode | -             | -      |
| icon     | 菜单项 icon，为 string 时会座位 Icon 组件的 name | string \| ReactNode | -             | -      |
| id       | 菜单项唯一标识                                   | string \| number    | -             | -      |
| disabled | 菜单项是否禁止点击                               | boolean             | true \| false | false  |
| children | 子菜单项配置                                     | DataItem[]          | -             | -      |
