# Charts 图表展示

- 基于开源组件[echarts](https://echarts.apache.org/zh/index.html)进行了主题定制
- 使用开源组件[echarts for react](https://github.com/hustcc/echarts-for-react)承担自定主题

## 使用方法

1. 安装 echarts && echarts-for-react

```shell
npm install echarts@4.9.0 echarts-for-react@2.0.16 --save
```

2. 引入 Charts 组件

```js
import Charts from '@hi-ui/hiui/es/charts' // 注意引用路径
```

3. 配置属性

```js
<Charts
  option={} // Echarts 官方配置项
  style={} // 设置图表宽高值
  className={} // 设置图表元素样式表
  showLoading={} // 是否显示加载状态
  // 更多配置
/>
```

3. Option
   [点此查看更多 echarts 图表属性](https://echarts.apache.org/zh/option.html)

4. Charts API
   [点此查看更多 Charts 属性](https://github.com/hustcc/echarts-for-react)
