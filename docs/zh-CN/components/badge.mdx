# Badge 红点 / 徽标

红点用来展示新消息的提示，徽标用于展示消息的数量、提示

## 何时使用

有新消息提示时

常见于图标、文字标题、按钮等位置

## Props

| 参数    | 说明                                     | 类型             | 可选值            | 默认值   |
| ------- | ---------------------------------------- | ---------------- | ----------------- | -------- |
| content | 气泡显示内容                             | string \| number | -                 | -        |
| type    | 气泡显示的形态，可选带文字的气泡或小红点 | string           | 'bubble' \| 'dot' | 'bubble' |
| max     | 气泡显示的最大值，超过值用'+'号替代      | number           | -                 | 99       |
| visible | 是否显示气泡                             | boolean          | true \| false     | true     |
| color   | 气泡颜色                                 | string           | -                 | -        |
| offset   | 设置状态点的位置偏移                                 | [left:number, top:number]           | -                 | -        |
